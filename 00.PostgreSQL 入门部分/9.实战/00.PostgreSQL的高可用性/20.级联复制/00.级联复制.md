# 级联复制
```
    PostgreSQL 备库不仅仅只支持所有的备库直连主库。实际上 PostgreSQL 支持备库既可以接收主库的WAL，也指出将 WAL 发送到其他的备库。这一特性称为级联复制。

    譬如：
        有一台主库 Master ，一台备库Slave01直连 Master。还有一台备库Slave02 连接 Slave01。
        这种架构下， Slave01 一方面接收 Master 发送的 WAL 日志，两一方面将 WAL 发送到 Slave02.
        这种既接收 WAL 日志 又发送 WAL 的备库称为级联备库。
        也将直连主库的备库称为上游节点，连接到上游节点的备库称为下游节点。
```
